# Body Upgrade — Кузница Тела

Мобильное веб-приложение для ведения силового цикла (3 тренировки/нед, 4-недельный цикл: 2 силовые + 2 объёмные недели) в фэнтези-сеттинге: герой, уровни, статы, вердикты «отработал / схалявил».

## Стек
- Чистые ES-модули, без сборщика — деплоится на Vercel как статика
- Программа цикла: `data/program.js`
- Журнал: localStorage + экспорт/импорт JSON-файла (кнопки в профиле)

## Локальный запуск
```bash
npx serve .
# или python3 -m http.server 8000
```

## Деплой
Пуш в main → Vercel собирает автоматически (Framework Preset: Other, без build command, output = корень).

## Логика вердикта
Счёт = 65% покрытие плановых подходов + 35% работа в своей весовой вилке (движение дня весит ×2).
- ≥85% — «Испытание пройдено»
- 60–84% — «Достойно, но не всё»
- <60% — «Схалтурил, странник»

XP → уровень героя. Статы: СИЛА (рост расчётных 1ПМ от базы 147/170/195/100), ВЫНОСЛИВОСТЬ (средний тоннаж), ДИСЦИПЛИНА (частота за 14 дней), СТОЙКОСТЬ (средний счёт).
